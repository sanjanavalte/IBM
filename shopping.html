Architecture

How ShopMind AI Works

An end-to-end agentic pipeline powered by IBM technologies, designed for speed, accuracy, and personalisation at scale.

1

User Intent Capture (Multimodal Input)

The user submits a query via text, voice, or product image. IBM Watson Speech-to-Text transcribes audio. Granite vision model decodes image-based product context. All inputs are normalized into a structured intent object.

IBM Watson + Granite Vision
2

Agentic Orchestration Layer (IBM Langflow)

IBM Langflow routes the intent to parallel tool-calling agents: one scrapes product data from e-commerce APIs, another fetches price history, a third runs review integrity analysis, and another checks sustainability databases.

IBM Langflow Multi-Agent Graph
3

Data Fusion & Reasoning (IBM Granite LLM)

IBM Granite synthesizes all fetched data — prices, reviews, specs, eco-scores, trend signals — into a ranked recommendation. It applies contextual filters (budget, urgency, sustainability preference) from the user's profile.

IBM Granite LLM Reasoning
4

Predictive Analytics & Alert Engine

A time-series model forecasts the price trajectory for each shortlisted product. If a drop is predicted, the agent schedules a proactive push alert via IBM Orchestrate automation. Real-time discount windows are scored by urgency.

IBM Orchestrate + Forecasting
5

Personalized Dashboard Output

Final results are rendered on the Smart Shopping Pulse dashboard — showing price comparisons, trust scores, eco rankings, trending items, and a personalized shopping list with one-click deep links to the best deal.

Real-Time React Dashboard
